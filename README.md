# FLaG

*FLaG* is abbreviation that corresponds to *Formal Languages and Grammars*.

# Prerequisites (Ubuntu/Debian only)

1. Install dotnet SDK 10.0. Follow the [instructions](https://dotnet.microsoft.com/en-us/download).
1. Install *texlive-full* with the following command.

```shell
$ sudo apt install texlive-full
```

# Installation (Ubuntu/Debian only)

1. Clone this repository.
1. Run the following command at the repository root to build.

```shell
$ dotnet build -c Release Sources/FLaG.slnx
```

# Execution (Ubuntu/Debian only)

1. Run the following command.

```shell
$ make
```
